# **App Name**: USM Marketplace4U

## Core Features:

- User Authentication: Secure user authentication system for registration, login, and logout.
- Item Management: Sellers can list, edit, and remove products with detailed descriptions and images.
- Search and Filter: Users can search for products based on keywords and filter by category, price, and condition.
- Admin Dashboard: Administrators can monitor user activity, manage listings, and handle disputes.
- Messaging: Buyers can contact sellers directly via WhatsApp using a provided link.
- AI Product Suggestion Tool: The app provides automatic listing descriptions to the user, with AI being used as a tool for generating high quality listing descriptions, leading to easier management.
- Database Integration: Firebase will be used to store product and user data, with Laravel's backend framework managing interactions.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to evoke trust and modernity.
- Background color: Light gray (#F5F5F5) for a clean and minimalist feel.
- Accent color: A bright green (#90EE90) to highlight key actions and elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif, provides a modern, machined, neutral look; suitable for both headlines and body text.
- Use a consistent set of simple, modern icons for navigation and product categories.
- Implement a clean and intuitive layout with a focus on easy navigation and clear information hierarchy.
- Incorporate subtle animations and transitions to enhance user experience and provide visual feedback.
